# Секретарь — как всё устроено

Шпаргалка, чтобы не путаться. Секретарь — это набор Python-движков на pcomp (`~/secretary/spawn/`), systemd-таймеры и два телеграм-бота.

## Два бота (главное, чтобы не путаться)

| Бот | Что делает | Как общаться |
|---|---|---|
| **@shikipassmacbot** | движки: меню, кнопки, доска, напоминания, команды | шлёшь `/команды`, жмёшь кнопки |
| **openclaw-бот** | Шики-разговор: утренние/вечерние брифы, обычный диалог | пишешь словами, отвечает Шики |

Разные боты специально: один — «пульт» (команды/кнопки), другой — «собеседник» (Шики).

## Команды (@shikipassmacbot)

Набери `/` — телеграм покажет список. Или `/help`.

```
/menu       меню-хаб со всеми разделами (кнопки)
/board      канбан-доска (проекты → колонки → карточки)
/comms      переписки: долги по ответам, обещания, дедлайны
/debt       кому не ответил (рабочий тг)
/day        якоря на сегодня (дедлайны/напоминания/расписание)
/hours      золотые часы — когда ты продуктивен
/routines   рутины (что сегодня не сделано)
/rem        активные напоминания
/year       год активности картинкой
/invest     инвест-баланс
/guard      инфра: диск / бэкап / мак
/help       этот список
```

## Меню-хаб

`/menu` шлёт одно сообщение с кнопками ко всем разделам. Тап → раздел прямо в том же сообщении, `‹ меню` назад. «Доска» разворачивается в полную навигацию, «Год» присылает картинку.

## Движки (что за что отвечает)

**Продуктивность**
- `board` — канбан: проекты/доски/колонки/карточки, дедлайны (календарь), метки, подзадачи, приоритеты, повторы, быстрый захват с макросами (`/board` или «добавь задачу …»).
- `routines` — привычки-рутины по слотам (утро/день/вечер), стрик.
- `reminders` — напоминания с кнопками [готово/+1ч/завтра].
- `anchors` — единый календарь якорей (дедлайны + напоминания + расписание).
- `golden_hours` — энергопрофиль по часам (git+zsh+трекер).

**Коммуникации**
- `comms` — единый читатель РАБОЧЕГО телеграма: долги по ответам, обещания, дедлайны, ghost. Наружу чужим не пишет — только черновики тебе. Личный тг не читается.
- `comms_export` — автоэкспорт рабочего тг каждые 20 мин (руками не надо).

**Отчёты/деньги**
- `reports` / `reports_img` — сухие отчёты + год-картинка «зелёных квадратиков».
- `invest` — ручной леджер инвест-баланса.
- `moneypath` — дистанция до первого платящего по проекту.

**Ритм/жизнь**
- `live` — утренний/вечерний бриф (Шики), лайф-трекинг.
- `planner` — вечерний план на завтра. `lastmile` — доведение до финиша. `diary` — дневник. `vent` — выговориться. `diet` — питание. `idea` — инбокс идей (текст/голос).

**Защита (M3)**
- `secret` — секреты в age-шифровании (`secret.get`), не plaintext.
- `redact` — вырезает токены/OTP из всего, что уходит в телегу/логи.
- `guard` — следит за диском/бэкапом/маком, алертит.

## Автоматика (таймеры)

```
reminders     каждые 30с — шлёт напоминания + ловит кнопки/команды
live          утро/вечер — брифы
guard         каждый час — диск/бэкап/мак
comms-export  каждые 20 мин — автоэкспорт рабочего тг
shiki-companion  M0 — следит за фокусом (заголовки окон)
```

## Приватность

- Личный телеграм (не рабочий) **не читается вообще**.
- Всё анализируется локально на pcomp; наружу к третьим лицам секретарь не пишет.
- Секреты зашифрованы (age), OTP/токены вырезаются перед отправкой.
